# Sistema de Bordo em C

Sistema de análise de leituras de sensores desenvolvido em C como trabalho acadêmico na UNISINOS.

## Funcionalidades
- Cálculo de média, mínimo e máximo entre três leituras
- Cálculo de desvios em relação à média
- Verificação de faixa aceitável com alerta visual colorido (ANSI)
- Barra gráfica proporcional à média
- Relatório completo com todas as análises
- Menu interativo com loop de simulações

## Tecnologias
- Linguagem C
- Interface por terminal com formatação ANSI colorida
